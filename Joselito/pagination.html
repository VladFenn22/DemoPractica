Overview

La paginación se crea con elementos HTML de lista para que los lectores de pantalla puedan anunciar la cantidad de enlaces disponibles. Utilice un elemento envolvente

para identificarlo como una sección de navegación para lectores de pantalla y otras tecnologías de asistencia.
